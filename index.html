Python Password Generator

Python Password Generator is a simple yet effective project designed to help users create strong, secure, and customizable passwords. In today's digital world, security is a top priority, and using weak or repetitive passwords can leave users vulnerable to cyber-attacks. This project aims to solve that problem by providing an easy way to generate complex passwords that are difficult to crack.

The Password Generator is built using Python, leveraging its random and string libraries to ensure randomness and variety in the generated passwords. The program prompts the user to input specific requirements such as desired password length and whether to include uppercase letters, lowercase letters, numbers, and special characters. Based on the user's choices, the generator assembles a dynamic character pool and randomly selects characters to construct a password that meets the selected criteria.

One of the key highlights of this project is its focus on both security and flexibility. The generated passwords are strong enough to resist brute-force and dictionary attacks, while the user-friendly interface makes it accessible even to non-technical users. Additionally, the code is designed to handle invalid inputs gracefully, making the user experience smooth and error-free.

Python Security User Interface

Currency Converter

The Currency Converter is a C++ console-based application designed to perform real-time currency conversions with accuracy and speed. The program allows users to select source and target currencies from a predefined list and input the amount for conversion. It applies fixed or user-updated exchange rates to calculate the converted value. The project focuses on using fundamental C++ concepts such as functions, switch-case statements, and user input validation. This Currency Converter is a practical tool for understanding real-world financial logic while reinforcing core programming principles like control structures, modular design, and data handling in C++.

Beyond its basic functionality, the Currency Converter enhances user interaction through a simple and intuitive interface. It is capable of handling invalid selections and inputs gracefully, ensuring the program remains stable and user-friendly. The code structure promotes easy maintainability, allowing developers to update exchange rates or add new currencies with minimal effort.

Working on this project helped me develop a deeper understanding of how financial transactions are handled in global systems, and how exchange rate calculations play a crucial role in real-world applications like banking, e-commerce, travel, and forex trading. It also offered a strong revision of mathematical operations, logical thinking, and C++ syntax.
